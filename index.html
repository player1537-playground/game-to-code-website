Teaching the future programming.

Everyday you use countless computers: to communicate, to learn, to share, and to play. Yet, less than a percent of Americans know how to program. This is a problem, leaving over 1,000,000 jobs not filled this year.

Start Learning Today!

The Importance of Programming

Programming is a tremendous part of society today, simplifying our lives while at the same time complicating them. As recent events have shown, programming is becoming a necessary skill. People must be able to protect their privacy, a rare luxury nowadays, but only one half of a percent (0.5%) of Americans know how to program and protect themselves.

Our Vision

In creating these sets of tutorials, we maintained the simple mantra of teaching programming through the use of computer games. Current educational methods are too complicated for a beginner to understand, resulting in a loss of interest or simply giving up.

Fun Stuff

When you learn to program you can do anything! You can make an automated coffee machine, a media center, or your own web server.

Contributing

We have a Github repository where you can get the latest copy of our code as well as contribute by submitting a pull request. Te current implementation features JavaScript and a simple templating system (which can be learned in a few minutes by examining existing levels). Don't know either of these yet? Don't worry! We will soon have a way for everyone to contribute, and until that happens, the tutorials themselves could always use some constructive criticism.
